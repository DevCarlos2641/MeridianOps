# MeridianOps

## 🚀 Acerca del Proyecto

Este es un sistema moderno desarrollado con **Angular**, aplicando las mejores prácticas y librerías de la industria. Su diseño está enfocado en garantizar un **alto rendimiento** y una excelente **experiencia de usuario (UX)**, mientras mantiene la **seguridad** y la **escalabilidad** como pilares fundamentales.

## 🏢 Soporte Multientidad (Sucursales/Propiedades)

El sistema está diseñado para operar no solo en una entidad, sino en **múltiples sucursales o propiedades**. Usted puede administrar y llevar el control operativo de **todas sus entidades** desde una única instancia del sistema.

## ✨ Características Principales

* **Arquitectura Modular:** El sistema está diseñado de forma modular, donde cada componente es independiente. Esto facilita el **desarrollo ágil**, el mantenimiento y previene la propagación de errores.
* **Seguridad Basada en Roles (RBAC):** La aplicación implementa un estricto control de acceso basado en roles, aumentando la **seguridad de la información** y asegurando que cada usuario solo acceda a las funcionalidades que le corresponden.

## 📦 Módulos Funcionales

### 1. Control de Inventario
Módulo central para la gestión completa del stock de mercancía.
* **Gestión de Movimientos:** Registro de altas, bajas y ajustes de inventario.
* **Reportes:** Generación de informes detallados sobre todos los movimientos realizados.
* **Permisos Restringidos:** Las operaciones críticas (altas, bajas y ajustes) están estrictamente limitadas a roles específicos para asegurar la integridad de los datos.

### 2. Gestión de Compras
Permite la administración de todo el ciclo de adquisición de mercancía.
* **Registro Completo:** Mantenimiento de un historial exhaustivo de todas las transacciones de compra.
* **Integración con Inventario:** Asigna automáticamente las altas en el inventario una vez que la compra es registrada y aprobada.

### 3. Asistencias (Biométrico/Fotográfico)
Sistema de registro de entrada y salida basado en identificación fotográfica.
* **Captura de Imagen:** Utiliza la cámara integrada del dispositivo o una cámara USB externa para la toma de asistencia.
* **Registro en Tiempo Real:** Guarda el registro de asistencia junto con la foto correspondiente de forma inmediata.

### 4. Gestión de Usuarios y Roles
Herramienta de administración para el personal y sus permisos dentro del sistema.
* **Gestión de Usuarios y Roles:** Creación, modificación y asignación de roles y permisos a los usuarios del sistema.
* **Control de Empleados:** Gestión del listado de empleados para la toma de asistencia y el control de su registro de uso de mercancía.
* **Visualización:** Listado de asistencias con la foto de registro respectiva.
